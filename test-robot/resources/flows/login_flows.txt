*** Keywords ***
Successful Login Flow
    [Arguments]    ${user}
    Start Browser Session
    Click Navbar Login Button
    Should Not Be Logged In
    Login As    ${user}
    Should Be Logged In As    ${user}
    Logout
    Close Browser

Failed Login Flow
    [Arguments]    ${user}
    Start Browser Session
    Click Navbar Login Button
    Login As    ${user}
    Page Should Contain Invalid Credentials Error
    Should Not Be Logged In
    Close Browser
